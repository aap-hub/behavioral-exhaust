# 7. Conclusion

We studied whether structural properties of agent reasoning traces are associated with task success in code repair tasks. Across 373 validated runs from two models (Claude Sonnet 4.6, Codex GPT-5.4) on 75 SWE-bench Lite tasks, we extracted 24 behavioral features at zero API cost and subjected them to within-repo correlation, Fisher-combined CWC decomposition, and within-repo permutation testing with 10,000 permutations.

Three findings emerged. First, behavioral exhaust carries signal, but that signal is model-specific. Only two structural features -- consecutive error streaks and early error rate -- show consistent associations with failure across both models. All linguistic features that survive rigorous testing are specific to one model, and one feature (contrastive marker density) flips polarity entirely: it is associated with success in Sonnet and failure in Codex within the same repository. Second, the strongest observational association is not a single feature but an interaction. In Sonnet, the combination of high deliberation length and high diagnostic precision in internal reasoning yields a 73% pass rate, while either feature alone yields 23-32%. This nonlinear structure would be invisible to monitoring systems that track features independently. Third, internal reasoning and agent messages carry structurally different information. The same word -- "actually" -- associates with failure in thinking blocks and success in agent messages, confirming a layer-specific dissociation that any feature extraction pipeline must respect.

These findings are associational. We have not demonstrated that a gate built on these features would improve agent reliability, nor have we evaluated prediction accuracy on held-out data. The natural next step is Phase 1 intervention testing: gating agent tool calls based on the model-specific features identified here and measuring whether gated runs outperform ungated runs using paired within-subject designs. Anthropic's published methodology for model evaluations -- clustered standard errors, pass^k metrics, and pre-registered power analysis -- provides the statistical framework for that test.

The contribution of this work is establishing that the behavioral exhaust agents already produce -- reasoning text structure, tool-call error patterns, thinking block content -- carries model-specific signal associated with task outcomes. This signal is free to extract, requires no prompt modification, and opens a measurement dimension complementary to existing approaches based on verbalized confidence or external critics. Whether that measurement dimension can be converted into a practical reliability improvement remains an open question for Phase 1.
